Web por dispositivo

Desktops/laptops

  • Distintas resoluciones
  • Diseñamos para la resolución más común del momento (800x600, 1024x768...)

Móviles/PDAs

  • Poca potencia, poca memoria
  • WAP/WML
  • XHTML-MP

Smartphones modernos/tablets

  • Tienen navegadores completos
  • Menos resolución de pantalla que un PC/laptop
  • Diseñamos para esa resolución
  • Distinto método de entrada (dedos vs ratón)
  • Diseñamos de acuerdo a esto (botones grandes, etc)

1. Con layouts fluídos

  width:800px;
}

  width:600px;
}
#aside {
  width:200px;
}
#container {
  max-width:800px;
}
#content {
  width:75%;
}
#aside {
  width:25%;
}

3. Con media queries

<link href="base.css" type="text/css" media="all" />
<link href="print.css" type="text/css" media="print" />
<link href="mobile.css" type="text/css" media="screen and (max-width:480px)" />
@media screen and (min-width:1280px) {
  #container {
    width:max-width:1280px;
  }
}
